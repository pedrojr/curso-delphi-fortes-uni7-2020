
Debugging
Marcar Breakpoint na linha selecionada: F5 (ou clicando do lado esquerdo do número da linha)
Executar/Continuar a aplicação em modo debug: F9
Executar sub-rotinas na linha atual: F7
Executar a linha atual: F8

Code Completion: Ctrl + Barra de Espaço
Code Parameters: Ctrl + Shift + Barra de Espaço

Indentar linhas selecionadas: Ctrl + Shift + I
Recuar linhas selecionadas: Ctrl + Shift + U

Ir para uma implementação: Ctrl + Clique Mouse

Se movimentar entre declaração de procedure/função e a implementação: Ctrl + Shift + Seta para cima ou Seta para baixo

Criar implementação depois de declarar procedure/função: Ctrl + Shift + C

Alterna entre maiúscula/minúscula um trecho selecionado: Ctrl + O + U

Seleção em colunas: Ctrl + Alt + Selecionar com o Mouse

Criar/remover Bookmark: Ctrl + Shift + 0 a 9

Ir para uma linha na unit: Alt + G
